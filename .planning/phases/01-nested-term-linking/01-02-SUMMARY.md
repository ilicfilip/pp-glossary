## Decisions Made

**Use wp_kses_post() instead of esc_html() for linked content**
- Rationale: Term_Linker generates `<a>` tags, so output needs to allow HTML
- Safe because link HTML is generated by our code, not user input
- wp_kses_post() allows standard post HTML including anchor tags

**Process descriptions through Term_Linker after long/short fallback**
- Rationale: Avoids processing both long and short descriptions unnecessarily
- More efficient - only process the description that will be displayed
- Cleaner code structure
